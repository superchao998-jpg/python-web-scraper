# Python Web Scraper

## Project Overview（项目简介）

This repository demonstrates a Python-based web scraping project designed to collect, process, and export publicly available web data.

It showcases practical solutions for web data extraction, data cleaning, Excel/CSV export, and automation workflows. The project serves as a portfolio to demonstrate my Python development and automation skills.
